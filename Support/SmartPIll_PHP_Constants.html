FM_DESKTOP_PATH

Description
The path to the desktop folder for the current user.
Examples
Mac OS:
/Users/User Name/Desktop/

Windows:
C:/Documents and Settings/User Name/Desktop/

FM_DOCUMENTS_PATH

Description
The path to the documents folder for the current user.
Examples
Mac OS:
/Users/User Name/Documents/

Windows XP:
C:/Documents and Settings/User Name/My Documents/

Windows Vista:
C:/Users/User Name/Documents/

FM_EXTENSIONS_PATH

Description
The path to the FileMaker extensions folder. This is the path where the plug-in is installed. Beginning with FileMaker 9, the extensions folder may be in the application folder as it was for previous versions or it may be in the user's local directory.
Examples
Mac OS:
/Applications/FileMaker Pro 8.5 Advanced/Extensions/

Mac OS (local):
/Users/User Name/Library/Application Support/FileMaker/Extensions/

Windows:
C:/Program Files/FileMaker/FileMaker Pro 8.5 Advanced/Extensions/

Windows XP (local):
C:/Document Settings/User Name/Local/Settings/ApplicationData/FileMaker/Extensions/

Windows Vista (local):
C:/Users/User Name/AppData/Local/FileMaker/Extensions/

FM_PHP_SUPPORT_PATH

Description
The path to the PHP support folder within the extensions folder.
Examples
Mac OS:
/Applications/FileMaker Pro 8.5 Advanced/Extensions/PHP Support/

Mac OS (local):
/Users/User Name/Library/Application Support/FileMaker/Extensions/PHP Support/

Windows:
C:/Program Files/FileMaker/FileMaker Pro 8.5 Advanced/Extensions/PHP Support/

Windows XP (local):
C:/Document Settings/User Name/Local/Settings/ApplicationData/FileMaker/Extensions/PHP Support/

Windows Vista (local):
C:/Users/User Name/AppData/Local/FileMaker/Extensions/PHP Support/

FM_PREFERENCES_PATH

Description
The path to the preferences and default options folder for the current user.
Examples
Mac OS:
/Users/User Name/Library/Preferences/

Windows XP:
C:/Document Settings/User Name/Local/Local Settings/Application Data/

Windows Vista:
C:/Users/User Name/AppData/Local/

FM_TEMPORARY_PATH

Description
The path to the temporary folder for the current user.
Examples
Mac OS:
/private/var/tmp/folders/uid/TemporaryItems/FileMaker/

Windows XP:
C:/Documents and Settings/User Name/Local Settings/Temp/

Windows Vista:
C:/Users/User Name/AppData/Local/Temp
